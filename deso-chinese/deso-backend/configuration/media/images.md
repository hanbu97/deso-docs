# Images

## GCP Credentials Path

`--gcp-credentials-path`

Type: String

Default: None

Path to google credentials necessary to upload to the bucket specified in [#gcp-bucket-name](images.md#gcp-bucket-name "mention")

## GCP Bucket Name

`--gcp-bucket-name`

Type: String

Default: None

Name of bucket to store images
